# 组件注册-总结

- **(1) Vue 组件在使用前需要先被“注册”，这样 Vue 才能在渲染模板时找到其对应的实现。**

# 组件注册

> 此章节假设你已经看过了[组件基础](/guide/essentials/component-basics)。若你还不了解组件是什么，请先阅读该章节。

一个 Vue 组件在使用前需要先被“注册”，这样 Vue 才能在渲染模板时找到其对应的实现。组件注册有两种方式：全局注册和局部注册。
